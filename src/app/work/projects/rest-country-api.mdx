## Overview

A comprehensive web application that displays information about countries worldwide using the REST Countries API. The application features advanced search and filtering capabilities, along with a modern dark/light theme system for enhanced user experience. Built with React and Tailwind CSS, this project showcases effective API integration and modern web development practices.

The application serves as an educational and informational tool that makes global geography data easily accessible and interactive. Users can explore detailed information about countries including demographics, geography, languages, and currencies through an intuitive search and filter system. The implementation demonstrates best practices for handling external API data, managing application state, and creating responsive user interfaces that work seamlessly across all devices.

## Key Features

- **Country Information**: Detailed data including population, capital, languages, and currencies
- **Search Functionality**: Real-time search by country name with instant results
- **Region Filtering**: Filter countries by continent/region
- **Dark/Light Theme**: Toggle between dark and light themes for user preference
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Pagination**: Efficient data loading with page-based navigation
- **Country Details**: Detailed view with additional information and borders
- **Loading States**: Smooth loading animations and error handling

## Technologies Used

- **React**: Frontend framework for building interactive components
- **REST API**: Integration with REST Countries API for data
- **Tailwind CSS**: Utility-first CSS framework for responsive design
- **JavaScript**: Dynamic functionality and API integration
- **Local Storage**: Theme preference persistence
- **Vercel**: Deployment platform for fast and reliable hosting

## API Integration

The application integrates with the REST Countries API to provide:
- Comprehensive country data
- Real-time search and filtering
- Efficient data caching
- Error handling and fallback states
- Responsive image loading

## User Experience Features

The application focuses on providing an intuitive user experience:
- Fast search with debounced input
- Smooth theme transitions
- Responsive grid layouts
- Accessible design patterns
- Mobile-first approach

## Challenges and Learnings

Working with external APIs required understanding data fetching patterns and error handling. Implementing efficient search and filtering while maintaining performance was a key challenge. Learning to manage application state and theme switching provided valuable insights into modern React development.

## Outcome

The REST Country API application successfully demonstrates API integration and modern frontend development practices. The responsive design and user-focused features create an effective platform for exploring global country information.
